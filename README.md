# 📡 Wi-Fi Analyzer Tool
<img width="546" height="498" alt="Screen Shot 2026-04-26 at 11 50 17" src="https://github.com/user-attachments/assets/db341593-e9ce-42af-adc8-3bff38183104" />

Простий GUI-інструмент на Python для моніторингу Wi-Fi сигналу в реальному часі (macOS).

## 🚀 Можливості

* Відображення поточної мережі (SSID)
* Візуалізація сили сигналу (%)
* Кольоровий індикатор якості:

  * 🟢 Сильний сигнал
  * 🟡 Середній
  * 🔴 Слабкий
* Логування історії сигналу
* Автоматичний запис у файл кожні ~5 секунд
* Попередження про слабкий сигнал
* Кнопки:

  * Відкрити папку з логами
  * Очистити логи

## 🖥️ Платформа

Працює тільки на **macOS**, оскільки використовує системну утиліту:

```
airport -I
```

## 📦 Залежності

Стандартні бібліотеки Python:

* `tkinter`
* `os`
* `time`
* `subprocess`

Нічого додатково встановлювати не потрібно.

## ▶️ Запуск

```bash
python wifi_analyzer.py
```

## 📁 Файл логів

Програма створює файл:

```
wifi_history.txt
```

Формат запису:

```
[12:34:56] Мережа: MyWiFi | Сигнал: 78%
```

## ⚠️ Особливості

* Оновлення кожну 1 секунду
* Запис у файл залежить від часу (може іноді дублюватися)
* Працює тільки якщо Wi-Fi увімкнений

## 🔧 Можливі покращення

* Підтримка Windows/Linux
* Графік сигналу
* Експорт у CSV
* Сповіщення (notification)

## 📸 Інтерфейс

Темна тема з простим UI:

* Назва мережі
* Великий відсоток сигналу
* Індикатор-бар
* Історія сигналу
